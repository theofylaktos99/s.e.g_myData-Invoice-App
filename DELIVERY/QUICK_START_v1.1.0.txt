📋 BEFORE YOU START
═══════════════════════════════════════════════════════════════════════════════
✓ Have you installed Node.js? If not → https://nodejs.org/ (LTS)
✓ Do you have AADE myDATA credentials? (User ID + 32-char key)
✓ Windows 10/11 recommended

🚀 3-STEP INSTALLATION
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Setup Backend (RUN ONCE!)
────────────────────────────────
1. Right-click "INSTALL-BACKEND-SERVICE.bat" 
2. Select "Run as Administrator"
3. Wait for "Setup complete!" message
⏱️ Takes: 2-3 minutes

STEP 2: Launch Application
──────────────────────────
1. Double-click "Italian Corner Invoice 1.0.0.exe"
2. App opens in a new window
⏱️ Takes: 10-15 seconds

STEP 3: Configure Settings (FIRST TIME ONLY)
─────────────────────────────────────────────
1. Click ⚙️ (Settings icon - bottom right)
2. Fill in:
   ├─ Backend URL: http://localhost:3000
   ├─ AADE User ID: (from myDATA Portal)
   ├─ Subscription Key: (from myDATA Portal)
   └─ GSIS Credentials: (optional, for VAT lookup)
3. Click "Έλεγχος σύνδεσης" (Test Connection)
4. Should see ✓ "Σύνδεση OK"

✅ DONE! Start creating invoices!


🎯 COMMON TASKS
═══════════════════════════════════════════════════════════════════════════════

LOOKUP CUSTOMER BY VAT
──────────────────────
1. Enter VAT number in "ΑΦΜ" field
2. Click "Αναζήτηση" (Search)
3. Customer details auto-fill from GSIS database

CANCEL AN INVOICE
─────────────────
1. Click "Ιστορικό" (History)
2. Find the invoice
3. Click "Ακύρωση" (Cancel)
4. Confirm cancellation

CHECK FOR UPDATES
─────────────────
1. Go to Settings ⚙️
2. The app checks automatically on startup
3. You'll see a notification if update available


⚠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

App won't start?
└─ Make sure Node.js is installed (node --version in Command Prompt)
└─ Make sure backend service is running (INSTALL-BACKEND-SERVICE.bat)

Backend service won't start?
└─ Open Command Prompt as Administrator
└─ Navigate to the folder
└─ Run: node aade-backend-standalone.cjs

GSIS lookup doesn't work?
└─ Check GSIS username/password in Settings
└─ Verify internet connection
└─ Try a known VAT: 135210960

Can't submit invoices?
└─ Check AADE credentials (User ID + Subscription Key)
└─ Click "Test Connection" to verify
└─ Try "Προπαραγωγή" (Preproduction) environment first

└─ Ελέγξτε τα credentials AAΔΕ (User ID + Subscription Key)
└─ Κάντε κλικ "Έλεγχος σύνδεσης"
└─ Δοκιμάστε πρώτα "Προπαραγωγή"
